# 二叉树

## 二叉树的前序遍历

### 题目

给定一个二叉树，返回它的 *前序* 遍历。

 **示例:**

```c++
输入: [1,null,2,3]  
   1
    \
     2
    /
   3 

输出: [1,2,3]
```

**进阶:** 递归算法很简单，你可以通过迭代算法完成吗？

### 题解

```c++
// 递归解法
// 先遍历当前节点，然后在遍历左节点，接着遍历右节点
class Solution {
public:
    vector<int> res;
    vector<int> preorderTraversal(TreeNode* root) {
        if (root != NULL) {
            res.push_back(root->val);
            preorderTraversal(root->left);
            preorderTraversal(root->right);
        }
        return res;
    }
};

// 非递归解法
class Solution {
public:
    vector<int> res;
    vector<int> preorderTraversal(TreeNode* root) {
        if (root == NULL) {
            return vector<int>();
        }
        stack<TreeNode *> st;
        st.push(root);
        while (!st.empty()) {
            TreeNode *ptr = st.top();
            st.pop();
            res.push_back(ptr->val);
            // 由于栈是逆序，所以这里应该先添加右节点，再添加左节点
            if (ptr->right != NULL) {
                st.push(ptr->right);
            }
            if (ptr->left != NULL) {
                st.push(ptr->left);
            }
        }
        return res;
    }
};
```

### 总结

- 递归和非递归两种方式

## 中序遍历二叉树

### 题目

给定一个二叉树，返回它的*中序* 遍历。

**示例:**

```c++
输入: [1,null,2,3]
   1
    \
     2
    /
   3

输出: [1,3,2]
```

**进阶:** 递归算法很简单，你可以通过迭代算法完成吗？

### 题解

```c++
// 递归方式
// 对于树中的任意节点来说，先打印它的左子树，然后再打印它本身，最后打印它的右子树。
class Solution {
public:
    vector<int> res;
    vector<int> inorderTraversal(TreeNode* root) {
        if (root != NULL) {
            inorderTraversal(root->left);
            res.push_back(root->val);
            inorderTraversal(root->right);
        }
        return res;
    }
};

// 迭代
class Solution {
public:
    vector<int> res;
    vector<int> inorderTraversal(TreeNode* root) {
        if (root == NULL) {
            return vector<int>();
        }
        stack<TreeNode *> st;
        TreeNode *ptr = root;
        /* 	1. 先到达最左边
			2. 把栈顶元素加入res中，然后把ptr指向栈顶元素的右节点
				2.1 如果右节点为nil，把ptr指向现在栈顶元素 
				2.2 如果右节点不为nil，把右节点压入栈中，一直查找到该节点的最后一个左节点
       	*/
        while (ptr != NULL || !st.empty()) {
            // 这里一进去，我们需要到达最左边节点，所以从根节点按顺序把节点压入栈中
            while (ptr != NULL) {
                st.push(ptr);
                ptr = ptr->left;
            }
            ptr = st.top();
            st.pop();
            res.push_back(ptr->val);
            ptr = ptr->right;
        }
        return res;
    }
};
```

## 后序遍历二叉树

### 题目

给定一个二叉树，返回它的 *后序* 遍历。

**示例:**

```c++
输入: [1,null,2,3]  
   1
    \
     2
    /
   3 

输出: [3,2,1]
```

**进阶:** 递归算法很简单，你可以通过迭代算法完成吗？

### 题解

```c++
// 递归方式
// 对于树中的任意节点来说，先打印它的左子树，然后再打印它的右子树，最后打印这个节点本身。
class Solution {
public:
    vector<int> res;
    vector<int> postorderTraversal(TreeNode* root) {
        if (root != NULL) {
            postorderTraversal(root->left);
            postorderTraversal(root->right);
            res.push_back(root->val);
        }
        return res;
    }
};

// 迭代
class Solution {
public:
    vector<int> postorderTraversal(TreeNode* root) {
       if (root == NULL) {
           return vector<int>();
        } 
        vector<int> res;
        stack<TreeNode *> st;
        TreeNode *ptr = root, *tag = NULL;
        while (ptr != NULL || !st.empty()) {
            // 到达二叉树最左边，并把沿途所有节点压入栈中
            // 每次当ptr指向右节点，把ptr和ptr的左节点全压入栈中
            while (ptr != NULL) {
                st.push(ptr);
                ptr = ptr->left;
            }
            ptr = st.top();
            st.pop();
            // 如果当前ptr右节点为NULL，把该ptr节点添加到res中，
            // 或者当tag == ptr->right，把该ptr节点添加到res中。
            // 如果当前ptr右节点不为NULL，把该ptr节点再次压入栈中
            if (ptr->right == NULL || tag == ptr->right) {
                res.push_back(ptr->val);
                // 用tag来标记遍历过的节点，如果该tag==ptr->right说明该ptr已经遍历过一次
                // 第一次遍历该ptr，由于右节点不为nil，所以把它再次压入栈中，遍历ptr的右节点
                // 当第二次指向该ptr，说明ptr的右节点说明已经遍历完了，所以就把该ptr添加到res
                tag = ptr;
                ptr = NULL;
            } else {
                st.push(ptr);
                ptr = ptr->right;
            }
        }
        return res;
    }
};
```

## 层次遍历二叉树

### 题目

给你一个二叉树，请你返回其按 **层序遍历** 得到的节点值。 （即逐层地，从左到右访问所有节点）。

**示例：**
二叉树：`[3,9,20,null,null,15,7]`,

```c++
    3
   / \
  9  20
    /  \
   15   7
```

返回其层次遍历结果：

```c++
[
  [3],
  [9,20],
  [15,7]
]
```

### 题解

```c++
class Solution {
public:
    vector<vector<int>> levelOrder(TreeNode* root) {
        vector<vector<int>> ve;
        if(root == NULL) return ve;
        
        queue<TreeNode *> qu;
        qu.push(root);
        while(!qu.empty()) { 
            vector<int> temp;
            // 这里qu的数量，就是每层的数量
            int level = qu.size();
            for (int i = 0; i < level; i++) {
                root = qu.front(); 
                qu.pop();
                temp.push_back(root->val);
                if(root->left != NULL) {
                    qu.push(root->left);
                }
                if(root->right != NULL) {
                    qu.push(root->right);
                } 
            }
            ve.push_back(temp);
        }
        return ve;
    }
};
```

## 二叉树的最大深度

### 题目

给定一个二叉树，找出其最大深度。

二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

**说明:** 叶子节点是指没有子节点的节点。

**示例：**
给定二叉树 `[3,9,20,null,null,15,7]`，

```c++
    3
   / \
  9  20
    /  \
   15   7
```

返回它的最大深度 3 。

### 题解

```c++
// 递归方式
class Solution {
public:
    int maxDepth(TreeNode* root) {
        if (NULL == root) {
            return 0;
        } else {
            return max(maxDepth(root->left), maxDepth(root->right)) + 1;
        }
    }
};
```

## 对称二叉树

### 题目

给定一个二叉树，检查它是否是镜像对称的。

 

例如，二叉树 `[1,2,2,3,4,4,3]` 是对称的。

```c++
    1
   / \
  2   2
 / \ / \
3  4 4  3
```

但是下面这个 `[1,2,2,null,3,null,3]` 则不是镜像对称的:

```c++
    1
   / \
  2   2
   \   \
   3    3
```

**进阶：**

你可以运用递归和迭代两种方法解决这个问题吗？

### 题解

```c++
// 递归解法
class Solution {
public:
    bool isSymmetric(TreeNode* root) {
        return isMirror(root, root);
    }
    
    bool isMirror(TreeNode* t1, TreeNode *t2) {
        // 判断两个节点都为空
        if (t1 == NULL && t2 == NULL) {
            return true;
        }
        // 其中一个节点为空，另一个不为空
        if (t1 == NULL || t2 == NULL) return false;
        // 每棵树的右子树都和左子树镜像对称
        return (t1->val == t2->val) 
            && isMirror(t1->left, t2->right) && isMirror(t1->right, t2->left);
    }
};

// 递归
// 初始化时我们把根节点入队两次。每次提取两个结点并比较它们的值（队列中每两个连续的结点应该是相等的，而且它们的子树互为镜像），
// 然后将两个结点的左右子结点按相反的顺序插入队列中。
// 当队列为空时，或者我们检测到树不对称（即从队列中取出两个不相等的连续结点）时，该算法结束。

class Solution {
public:
    bool isSymmetric(TreeNode* root) {
        return isMirror(root, root);
    }
    
    bool isMirror(TreeNode* t1, TreeNode *t2) {
        queue<TreeNode *> qu;
        qu.push(t1);
        qu.push(t2);
        while (!qu.empty()) {
            t1 = qu.front(); qu.pop();
            t2 = qu.front(); qu.pop();
            if (!t1 && !t2) {
                continue;
            }
            if ((!t1 || !t2) || (t1->val != t2->val)) {
                return false;
            }
            qu.push(t1->left);
            qu.push(t2->right);
            
            qu.push(t1->right);
            qu.push(t2->left);
        }
        return true;
    }
};
```

## 路径总和

### 题目

给定一个二叉树和一个目标和，判断该树中是否存在根节点到叶子节点的路径，这条路径上所有节点值相加等于目标和。

**说明:** 叶子节点是指没有子节点的节点。

**示例:** 
给定如下二叉树，以及目标和 `sum = 22`，

```c++
              5
             / \
            4   8
           /   / \
          11  13  4
         /  \      \
        7    2      1
```

返回 `true`, 因为存在目标和为 22 的根节点到叶子节点的路径 `5->4->11->2`。

### 题解

```c++
// 递归方式
class Solution {
public:
    bool hasPathSum(TreeNode* root, int sum) {
        // 递归跳出条件，指向NULL了
        if (root == NULL) {
            return false;
        }
        // 每次减去当前指针的val
        sum -= root->val;
        if (root->left == NULL && root->right == NULL) {
            return sum == 0 ? true : false;
        }
        // 先左后右，只要有一个满足即可
        return hasPathSum(root->left, sum) || hasPathSum(root->right, sum);
    }
};

// 非递归方式
class Solution {
public:
    bool hasPathSum(TreeNode* root, int sum) {
        if (root == NULL) {
            return false;
        }
        stack<TreeNode *> nodeStack; // 记录节点
        stack<int> sumStack; // 记录当前节点的sum
        nodeStack.push(root);
        sumStack.push(sum - root->val);
        
        TreeNode *ptr = NULL;
        int curSum = 0;
        while (!nodeStack.empty()) {
            ptr = nodeStack.top(); nodeStack.pop();
            curSum = sumStack.top(); sumStack.pop();
            if (ptr->left == NULL && ptr->right == NULL && curSum == 0) {
                return true;
            }
            
            if (ptr->right != NULL) {
                nodeStack.push(ptr->right);
                sumStack.push(curSum - ptr->right->val);
            }
            
            if (ptr->left != NULL) {
                nodeStack.push(ptr->left);
                sumStack.push(curSum - ptr->left->val);
            }
        }
        return false;
    }
};
```
